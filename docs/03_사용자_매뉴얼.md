## 1. 시작하기

### 1.1 접속 방법

웹 브라우저에서 아래 주소로 접속합니다:

- **로컬 개발:** `http://localhost:3000`
- **운영 서버:** 관리자가 안내하는 주소

> 💡 **지원 브라우저:** Chrome, Edge, Safari, Firefox (최신 버전)

### 1.2 로그인

1. 접속하면 **로그인 화면**이 나타납니다.
2. 이메일과 비밀번호를 입력하고 **로그인** 버튼을 클릭합니다.
3. 빠른 접속이 필요하면 하단의 **Quick Access** 버튼으로 데모 계정으로 즉시 로그인할 수 있습니다.

| 데모 계정 | 이메일 | 비밀번호 |
|----------|--------|----------|
| 관리자 | admin@baikal.ai | admin1234 |
| 김철수 | kim@baikal.ai | user1234 |
| 이영희 | lee@baikal.ai | user1234 |

### 1.3 로그아웃

- 좌측 사이드바 하단의 **로그아웃** 버튼을 클릭합니다.
- 세션이 만료되면(8시간) 자동으로 로그인 화면으로 이동합니다.

로그인 후 **3단 레이아웃**으로 구성됩니다:

```
┌─────────────┬──────────────────────────────┬────────────────┐
│             │                              │                │
│  사이드바    │      메인 콘텐츠 영역         │  AI 채팅 패널   │
│  (메뉴)     │  (대시보드/결재/업무 등)       │                │
│             │                              │                │
│  ┌───────┐  │                              │  ┌──────────┐  │
│  │대시보드│  │                              │  │ 메시지... │  │
│  │전자결재│  │                              │  │          │  │
│  │업무관리│  │                              │  │          │  │
│  │일정관리│  │                              │  │  입력창   │  │
│  │공지사항│  │                              │  └──────────┘  │
│  └───────┘  │                              │                │
│             │                              │                │
│  로그아웃    │                              │                │
└─────────────┴──────────────────────────────┴────────────────┘
```

- **사이드바:** 메뉴 네비게이션, 사용자 프로필, 로그아웃
- **메인 콘텐츠:** 선택한 메뉴의 내용

### 2.2 대시보드

로그인 후 처음 보이는 화면으로 한눈에 현황을 파악할 수 있습니다:

- **통계 카드 (4개):** 결재 대기, 진행 업무, 오늘 일정, 공지사항 수
- **최근 활동:** 최근 결재/업무/일정/공지사항 미리보기
- **AI 배너:** AI 기능 안내

## 3. AI 어시스턴트 사용법

### 3.1 AI란?

BAIKAL Groupware AI의 **핵심 기능**입니다. 메뉴를 찾아 클릭할 필요 없이, **자연어로 말하면** AI가 자동으로 실행합니다.

### 3.2 사용 방법

1. 우측의 **AI 채팅 패널**에서 메시지를 입력합니다.
2. 원하는 작업을 자연스럽게 한국어로 작성합니다.
3. AI가 자동으로 작업을 수행하고 결과를 알려줍니다.

### 3.3 AI에게 할 수 있는 것

#### 📋 전자결재
```
"출장 신청서 만들어줘"
"서울 출장 결재 올려줘, 내용은 고객 미팅 참석"
"결재 목록 보여줘"
"내가 올린 결재 현황 알려줘"
```

#### ✅ 업무 등록
```
"김철수에게 보고서 작성 업무 등록해줘"
"이영희에게 긴급 업무 할당. 제목: 버그 수정, 마감: 3월 15일"
"내 업무 목록 보여줘"
```

#### 📅 일정 등록
```
"3월 10일 오후 2시 회의 등록해줘"
"내일 오전 10시부터 11시 팀 미팅, 장소는 3층 회의실"
"이번 주 일정 보여줘"
```

#### 📢 공지사항
```
"회사 워크숍 공지 작성해줘"
"전직원 대상 보안 교육 공지 올려줘"
"공지사항 목록 보여줘"
```

#### 👥 사용자 검색
```
"김철수 찾아줘"
"개발팀 사람 검색해줘"
```

### 3.4 추천 문장

채팅 패널 상단에 **추천 문장 버튼**이 있습니다. 클릭하면 바로 AI에게 전송됩니다:

- 📋 결재 올려줘
- ✅ 업무 등록해줘
- 📅 회의 잡아줘
- 📢 공지 작성해줘

### 3.5 결과 확인

AI가 작업을 완료하면:
1. **채팅 메시지**로 결과를 알려줍니다.
2. 우측에 **결과 패널**이 나타나 상세 내용을 보여줍니다.
3. 해당 페이지(전자결재, 업무관리 등)에서도 확인할 수 있습니다.

## 4. 전자결재

### 4.1 결재 문서 보기

1. 사이드바에서 **전자결재**를 클릭합니다.
2. 결재 문서 목록이 표시됩니다.
3. **필터 탭**으로 상태별 조회가 가능합니다:
   - 전체 / 초안 / 대기 / 승인 / 반려

### 4.2 결재 문서 만들기

**방법 1 — AI로 만들기 (추천):**
```
AI: "출장 신청서 만들어줘. 목적: 부산 고객 미팅, 일정: 3월 10-12일"
```

**방법 2 — 직접 만들기:**  
현재 MVP에서는 AI를 통한 생성만 지원합니다.

### 4.3 결재 상신 (제출)

1. 상태가 **초안**인 문서의 **상신** 버튼을 클릭합니다.
2. 문서가 **대기** 상태로 변경되어 결재라인의 결재자에게 전달됩니다.

### 4.4 결재 승인 / 반려

1. 상태가 **대기**인 문서에서 **승인** 또는 **반려** 버튼을 클릭합니다.
2. (선택) 코멘트를 입력합니다.
3. 결재라인 순서대로 결재가 진행됩니다.

### 4.5 상태 설명

| 상태 | 색상 | 설명 |
|------|------|------|
| 초안 | 회색 | 작성 완료, 아직 제출하지 않음 |
| 대기 | 노란색 | 결재자 승인 대기 중 |
| 승인 | 초록색 | 결재 완료 |
| 반려 | 빨간색 | 결재 반려됨 |

### 4.6 결재 문서 상세

문서를 클릭하면 **상세 팝업**이 열립니다:
- 제목 / 카테고리 / 작성자 / 상태
- 문서 내용
- 결재라인 (결재 순서, 결재자명, 처리 상태)

## 5. 업무관리

### 5.1 칸반 보드

업무관리 페이지는 **3단 칸반 보드**로 구성되어 있습니다:

| 컬럼 | 설명 |
|------|------|
| **To Do** | 아직 시작하지 않은 업무 |
| **진행중** | 현재 진행 중인 업무 |
| **완료** | 완료된 업무 |

### 5.2 업무 만들기

**AI로 만들기 (추천):**
```
"김철수에게 월간 보고서 작성 업무 등록해줘. 우선순위 높음, 마감 3월 31일"
```

### 5.3 업무 상태 변경

각 업무 카드에서:
- **To Do** → **시작** 버튼 → 진행중으로 이동
- **진행중** → **완료** 버튼 → 완료로 이동

### 5.4 업무 카드 정보

각 카드에 표시되는 정보:
- **제목**: 업무명
- **우선순위**: 낮음 · 보통 · 높음 · 긴급
- **담당자**: 아바타 + 이름
- **마감일**: 날짜 표시
- **설명**: 업무 상세 설명 (있는 경우)

### 5.5 우선순위

| 우선순위 | 표시 | 설명 |
|---------|------|------|
| 낮음 | ● 회색 | 여유 있는 업무 |
| 보통 | ● 파란색 | 일반적 업무 |
| 높음 | ● 노란색 | 중요한 업무 |
| 긴급 | ● 빨간색 | 즉시 처리 필요 |

## 6. 일정관리

### 6.1 일정 보기

일정관리 페이지는 **타임라인 뷰**로 날짜별로 그룹화되어 표시됩니다:
- 왼쪽에 **날짜 카드**: 월/일, 요일 표시
- 오른쪽에 **일정 목록**: 시간, 장소, 설명

> 💡 **오늘 날짜**는 초록색 그라데이션으로 강조 표시됩니다.

### 6.2 일정 등록

**AI로 등록 (추천):**
```
"내일 오후 3시부터 4시 팀 미팅 등록해줘. 장소는 5층 대회의실"
"3월 15일 10:00~11:00 고객 미팅, 장소: 1층 로비"
```

### 6.3 일정 정보

각 일정 카드에 표시되는 정보:
- **제목**: 일정명
- **시간**: 시작 ~ 종료 시간
- **장소**: 위치 정보 (있는 경우)
- **설명**: 추가 설명 (있는 경우)

## 8. 모바일 사용법

BAIKAL Groupware AI는 **모바일 반응형**으로 설계되었습니다. 스마트폰, 태블릿에서도 모든 기능을 사용할 수 있습니다.

### 8.1 모바일 화면 구성

모바일에서는 화면이 최적화됩니다:

```
┌──────────────────────┐
│  ☰  BAIKAL    💬    │  ← 상단 바 (햄버거 메뉴 / 채팅 토글)
├──────────────────────┤
│                      │
│   메인 콘텐츠 영역    │
│                      │
│                      │
│                      │
│                      │
│                      │
│        ● 💬          │  ← 우측 하단 플로팅 AI 채팅 버튼
└──────────────────────┘
```

### 8.2 메뉴 열기

- 좌측 상단 **☰ 햄버거 버튼**을 탭합니다.
- 사이드바 메뉴가 슬라이드되어 나타납니다.
- 메뉴 선택 후 자동으로 닫힙니다.
- 배경을 탭해도 닫힙니다.

### 8.3 AI 채팅

- 우측 하단 **보라색 원형 버튼(💬)**을 탭합니다.
- AI 채팅 패널이 하단에서 슬라이드업됩니다.
- PC와 동일한 기능을 모두 사용할 수 있습니다.

### 8.4 모바일 최적화 기능

| 기능 | PC | 모바일 |
|------|----|----|
| 전자결재 | 테이블 뷰 | 카드 리스트 뷰 |
| 업무관리 | 3단 칸반 보드 | 탭 전환 1단 뷰 |
| 일정관리 | 넓은 타임라인 | 컴팩트 타임라인 |
| 공지사항 | 리스트 뷰 | 동일 (반응형) |

## 9. 자주 묻는 질문 (FAQ)

### Q1. AI가 이해하지 못할 때는 어떻게 하나요?
> 더 구체적으로 말해보세요.  
> ❌ "결재" → ✅ "출장 결재 문서 만들어줘"  
> ❌ "업무" → ✅ "김철수에게 보고서 업무 등록해줘"

### Q2. AI가 잘못된 작업을 했어요.
> 현재 MVP에서는 AI가 실행한 작업의 취소 기능은 제공하지 않습니다.  
> 수동으로 해당 메뉴에서 수정하거나, 관리자에게 문의하세요.

### Q3. 비밀번호를 변경하고 싶어요.
> 현재 MVP에서는 비밀번호 변경 UI가 제공되지 않습니다.  
> 관리자에게 요청하세요.

### Q4. 결재 문서를 수정할 수 있나요?
> **초안** 상태의 문서만 수정 가능합니다.  
> **대기** 이후 상태의 문서는 수정할 수 없습니다.

### Q5. 다른 사람의 업무를 볼 수 있나요?
> 네, 전체 업무 목록은 모든 사용자에게 표시됩니다.  
> "내 업무"는 AI에게 "내 업무 보여줘"라고 요청하면 필터링됩니다.

### Q6. 오프라인에서도 사용 가능한가요?
> 아니요, BAIKAL Groupware AI는 온라인 환경에서만 사용 가능합니다.

### Q7. 어떤 브라우저를 사용해야 하나요?
> Chrome, Edge, Safari, Firefox의 최신 버전을 권장합니다.  
> Internet Explorer는 지원하지 않습니다.

### Q8. AI 대화 기록은 저장되나요?
> 네, 대화 기록은 서버에 저장되며 다시 로그인해도 이전 대화가 유지됩니다.
